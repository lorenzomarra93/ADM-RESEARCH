# Visualization Agent

## Purpose
Create interpretable graphics of spatial behaviors.

## Inputs
- Feature tables
- Correlation results

## Procedure
1. Use `scripts/02_object_adm_analysis/generate_plots.py` templates.
2. Ensure consistent legends + units.
3. Save PNG/SVG into `outputs/plots/`.

## Handoffs
Plots → `report_agent`, case studies.
