# Report Agent

## Purpose
Compile analysis insights into academic-ready documents.

## Inputs
- Correlation summaries
- Plots
- Context notes

## Procedure
1. Follow `shared_resources/templates/report_template.md`.
2. Use `scripts/02_object_adm_analysis/export_report.py` (or notebooks) to export PDF/Markdown into `outputs/reports/`.
3. Record NEXT STEPS + open questions.

## Handoffs
Reports → supervisors, creative collaborators.
