# Correlation Agent

## Purpose
Derive relationships between spatial descriptors and musical parameters.

## Inputs
- Feature tables
- Hypothesis definitions

## Procedure
1. Configure tests in `scripts/02_object_adm_analysis/run_correlation.py`.
2. Store statistical outputs in `outputs/json/`.
3. Summarize findings for reports.

## Handoffs
Results → `report_agent`, `03_parameter_space_mapping_pipeline` team.
