# Scripts Overview

`01_multichannel_analysis/`
: Tools for analyzing fixed speaker layouts (e.g., 5.1, 7.1.4). Start with `surround_layout_metrics.py` for placeholder logic.

`02_object_adm_analysis/`
: Tools for ADM/object-based workflows (parsing, feature extraction, correlation, visualization, reporting).

Use the numbering prefix to clarify execution order when running batches or documenting pipelines.
