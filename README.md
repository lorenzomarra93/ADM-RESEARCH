# ADM RESEARCH REPOSITORY

Immersive audio PhD workspace organized around two macro-areas:
- `02_analysis_pipeline` handles ingestion, parsing, and quantitative reporting of ADM/BW64/object-based recordings.
- `03_parameter_space_mapping_pipeline` develops compositional strategies, control mappings, and evaluation protocols.

Use `MAIN_INSTRUCTION.md` for governance, `PROJECT_ROADMAP.md` for planning, and `shared_resources/` for templates and bibliographic assets. Existing legacy folders remain untouched until their contents migrate into the new layout.
