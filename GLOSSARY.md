# Glossary

| Term | Definition |
| --- | --- |
| ADM | Audio Definition Model, metadata standard describing objects, beds, and scene-based audio. |
| BW64 | Broadcast Wave 64 format that encapsulates ADM metadata alongside multichannel PCM. |
| Object-Based Audio | Rendering paradigm controlling discrete sound objects with metadata rather than fixed channels. |
| Parameter-Space Mapping | Method that links musical parameters (form, harmony, gesture) to spatial control dimensions. |
| OSC | Open Sound Control protocol for transmitting musical/spatial control data. |
| ADM-OSC | Mapping between ADM metadata and OSC messages for interactive control or rendering. |
| Micro/Meso/Macro | Structural scales used to frame analytical and compositional decisions. |
| Feature Extraction | Computation of descriptors (energy, centroid, divergence) from audio/metadata streams. |
| Spatial Timeline | Representation of spatial events aligned with musical form for playback or notation. |
| Agent | Focused instruction set describing how to execute a task or workflow autonomously. |
